# Object Oriented Programming

## Classes

### Class Definition

```python
class Person:
    pass
```

### Class Instantiation

```python
person = Person()
```

### Class Attributes

```python
class Person:
    name = "John"
    age = 36
```

### Class Methods

```python
class Person:
    name = "John"
    age = 36

    def greet(self):
        print(f"Hello, my name is {self.name}")
```
